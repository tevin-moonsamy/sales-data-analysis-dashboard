# Sales Data Analysis Dashboard (Excel Project)

## Project Overview
This project is a sales data analysis dashboard built using Microsoft Excel. The goal is to analyze retail sales performance and generate actionable business insights from raw transactional data.

The dashboard identifies top-performing products, monthly sales trends, and revenue distribution across regions and categories.

## Objectives
- Clean and structure raw sales data  
- Analyze key business metrics  
- Identify top-performing products  
- Track monthly sales trends  
- Break down revenue by category and region  
- Build an interactive Excel dashboard using pivot tables and slicers  

## Tools and Technologies
- Microsoft Excel  
- Pivot Tables  
- Data Cleaning Functions  
- Charts and Graphs  
- Slicers (Interactive Filters)  

## Dataset Description
The dataset includes the following fields:
- Order ID  
- Date  
- Product Name  
- Category  
- Region  
- Quantity Sold  
- Unit Price  
- Total Sales (calculated field)  

## Data Preparation
The following data cleaning steps were performed:
- Converted raw data into structured Excel tables  
- Created a calculated field for Total Sales (Quantity × Unit Price)  
- Extracted Month from Date for trend analysis  
- Ensured consistent formatting across all columns  

## Key Analysis Performed

Top Products  
- Identified highest revenue-generating products using pivot tables  

Monthly Sales Trends  
- Analyzed sales performance over time using date aggregation  

Revenue by Category  
- Compared performance across product categories  

Revenue by Region  
- Evaluated sales distribution across different regions  

## Dashboard Features
- Interactive Excel dashboard  
- Pivot charts for visual analysis  
- Slicers for filtering by Region and Category  
- Dynamic data exploration
